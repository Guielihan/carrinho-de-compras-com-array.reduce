# Imagens dos Produtos - Carrinho de Compras

## Instruções

Substitua os arquivos PNG abaixo pelos respectivos produtos:

### Arquivos necessários:

1. **notebook.png** - Imagem do Notebook (2500 BRL)
2. **mouse.png** - Imagem do Mouse (89,90 BRL)
3. **teclado-mecanico.png** - Imagem do Teclado Mecânico (350 BRL)
4. **monitor-27.png** - Imagem do Monitor 27" (1200 BRL)
5. **webcam-hd.png** - Imagem da Webcam HD (250 BRL)
6. **headset-gamer.png** - Imagem do Headset Gamer (450 BRL)
7. **ssd-1tb.png** - Imagem do SSD 1TB (400 BRL)
8. **memoria-ram-16gb.png** - Imagem da Memória RAM 16GB (300 BRL)

### Recomendações:

- Formato: PNG com fundo transparente
- Tamanho recomendado: 500x500px ou maior (serão redimensionadas para 120x120px)
- Qualidade: Alta resolução para melhor visualização
- Estilo: Profissional e consistente

### Como fazer upload:

1. Adicione cada arquivo PNG nesta pasta (`assets/`)
2. Use exatamente os nomes listados acima
3. Faça um commit com as imagens
4. Faça push para o GitHub

As imagens aparecem com fallback automático (placeholder cinza) se não encontradas.
